# Deep City Verona

Landing page for a curated tourism project based in Verona, focused on exclusive and immersive experiences beyond standard tours.

## Overview

Deep City Verona presents a selection of high-end experiences in the city, including romantic tours, historical walks, food and wine tastings, and VIP access activities.

The goal is to communicate a more personal and story-driven way of exploring Verona, combining strong visuals with clear conversion-focused structure.

## Features

- Experience catalog with categories (Romantic, Historic, Gastronomic, VIP)
- Detailed experience pages/cards with duration, pricing, and descriptions
- Bespoke section for custom experiences
- Testimonials / traveler reviews
- Image gallery of Verona highlights
- Booking/contact form with structured inputs
- FAQ section

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Structure

- Hero section with main branding and CTA
- Experiences section with filters
- About / Story section
- Tailor-made (bespoke experiences)
- Gallery
- Testimonials
- Booking form
- Footer with navigation and legal links

## Purpose

This project was built as a portfolio piece to practice UI/UX design, layout structuring, and creating a conversion-oriented landing page for a real-world business concept.
